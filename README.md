<h1 align="left">Free Roblox Bundle Buyer</h1>

<p align="left">Automatically buys new or previously free bundles in the Roblox catalog.</p>

### Usage
Update the config.json accordingly 
```json
{
    "accounts": {
        "main_account": "",
        "alt_accounts": []
    },
    "webhook": {
        "enabled": true,
        "url": ""
    },
    "misc": {
        "bundles": true,
        "only_new": false
    }
}
```
